Objetivos generales

Bienvenidos a el modulo de informacion de git y GitHub GIT: Un Software de control, se define como control de versiones de la gestion de los diversos cambios que se pueden realizar sobre elementos de los diferentes productos, tambien sirve para desarrollar un software o una pagina web. Git Fue creado pensando en la eficiencia y la confiabilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran numero de archivos de codigo fuente, git nos proporciona herramientas para desarrollar nuestros trabajos en equipo de manera inteligente y rapida. GitHub: es un sistema de gestión de proyectos y control de versiones de código, así como una plataforma de red social diseñada para desarrolladores. ¿Pero para qué se usa GitHub? Bueno, en general, permite trabajar en colaboración con otras personas de todo el mundo, planificar proyectos y realizar un seguimiento del trabajo. Git: Antes de continuar, debemos entender a Git como el corazón de GitHub. Git es un sistema de control de versiones desarrollado por Linus Torvalds (el hombre que creó Linux). Hub: Si Git es el corazón de GitHub, entonces Hub es su alma. El hub de GitHub es lo que convierte una línea de comandos, como Git, en la red social más grande para desarrolladores. Además de contribuir a un determinado proyecto, GitHub le permite a los usuarios socializar con personas de ideas afines. Puedes seguir a las personas y ver qué hacen o con quién se conectan.
