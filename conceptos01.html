GIT


¿Que es GIT?


Git es una herramienta para realizar una funcion de control de versiones de codigo muy potente la cual y distribulle la informacion de manera rapida, git nos proporciona las herramientas para poder desarrollar nuestro trabajo ya sea software, una pagina web, cualquier aplicacion que implique usar un codigo usando un equipo de manera inteligente y rapida, al ser en linea podemos compartir nuestro codigo con nuestros compañeros


Las caracteristicas importantes de Git son:


  • Gestion distribuida, los cambios realizados en el proyecto pueden ser fusionados de manera rapida y sencilla
  • Realizar seguimientos a tiempo real de nuestro codigo
  • Al usar un repositorio local, se puede enviar cambios en el codigo sin necesidad de internet

    • git cuenta con ciertas ordenes basicas en el codigo, las cuales son:


    ORDENES BASICAS


  • Iniciar un nuevo repositorio vacio en una carpeta especifica

  • Añadir un nuevo archivo, con un nombre especifico

  • Añadir todo los archivos del directorio

  • Confirmar los cambios realizados

  • Revertir los cambios realizados

  • Subir la rama al servidor remoto

  • Mostrar el estado actual de la rama
